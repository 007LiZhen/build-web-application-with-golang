# 1.1 GO 安装

## Window 安装
    
  访问[下载地址](downlink),32位机器下载go1.0.2.windows-386.msi，64位机器下载go1.0.2.windows-amd64.msi，双击下载文件，一路按照默认点击下一步，这个时候go已经安装到你的系统中，默认安装之后已经在你的系统环境变量中加入了c:/go，这个时候打开CMD，输入
	
	go
